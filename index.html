Acerca de esta herramienta

Es una herramienta simple para crear una línea del tiempo básica con los elementos principales que la caracterizan.

Recomendaciones y datos

Se recomienda utilizar imágenes del mismo tamaño para seguir un orden y los elementos no se desborden.

Puedes cambiar el estilo de diferentes elementos dando clic en el icono de configuración en el futuro se añadirán más funciones para una mejor personalización.

Puedes colocar los eventos que desees, además, los títulos y el texto no tiene un límite.
